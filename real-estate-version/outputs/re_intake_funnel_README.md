# Real Estate Intake Funnel — Sunrise Realty Group (demo)

Clone of the PI-law AI intake funnel, real-estate edition. Generated by the
`realestate-intake-funnel-builder` skill — **never hand-edit these JSONs or edit them on the
n8n canvas**; change `demo_config.json` or the skill templates and rebuild.

## Files

| File | Role |
|---|---|
| `re_intake_funnel_workflow.json` | Landing-page lead → Claude qualifies (qualified / needs_info / declined) → returns slots or books on Google Calendar → texts lead → responds. Webhook: `POST /webhook/re-intake` |
| `re_inbound_bot_workflow.json` | Two-way WhatsApp bot (AI Agent + window memory) — qualifies in-thread, offers times in plain language, books deterministically, replies via TwiML. Webhook: `POST /webhook/re-inbound` |
| `re_intake_landing.html` | Landing page; form POSTs to `/webhook/re-intake`, slot picker books, WhatsApp deep-link continues to the bot |
| `demo_config.json` | Build input (agency, calendar, Twilio, criteria, n8n base URL) |

## Import (each workflow, FRESH — never paste onto an existing canvas)

1. n8n → Workflows → **Import from File**.
2. Funnel credentials: Anthropic (`httpHeaderAuth`, header `x-api-key`), Twilio (`twilioApi`),
   Google Calendar OAuth. (Sheets logging disabled in this config.)
3. Bot credentials: Anthropic (`anthropicApi`) + Google Calendar only.
4. **Activate both** (bot memory/static data only persists on active production runs).
5. Twilio WhatsApp sandbox → "When a message comes in" → `https://josesn8n.win/webhook/re-inbound` (POST).

## Rebuild

```bash

  outputs/demo_config.json \
  --out outputs/re_intake_funnel_workflow.json \

```

## ⛔ Compliance (non-negotiable)

The AI qualifies and schedules ONLY. Never gives valuations, price opinions, or market
predictions; never gives legal/tax/lending advice; never guarantees outcomes; no agency
relationship formed; **Fair Housing enforced** (no demographic/steering answers — declines and
redirects to consult). Enforced in the funnel's `Set: AI Config` node and the bot's agent
system message — keep intact through every rebuild.
